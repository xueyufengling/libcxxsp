# libcxxsp

    libcxxsp提供了使用汇编和C++实现的机器、操作系统相关的底层操作函数，同时提供了一套GCC语法的汇编宏用于简化内联汇编的编写。需要不低于C++11的标准。该库依赖于预处理器元编程[libppmp](https://github.com/xueyufengling/libppmp)、模板元编程[libtplmp](https://github.com/xueyufengling/libtplmp)以及[libllvmmci](https://github.com/xueyufengling/libllvmmci)。<br>
