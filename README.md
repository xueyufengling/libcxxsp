# libcxxsp

    libcxxsp provides low-level machine and operating system related operation functions implemented in assembly and C++, and also provides a set of GCC syntax assembly macros to simplify the writing of inline assembly. Standard not lower than C++11 is required. This library depends on preprocessor meta-programming [libppmp](https://github.com/xueyufengling/libppmp) and template meta-programming [libtplmp](https://github.com/xueyufengling/libtplmp).
